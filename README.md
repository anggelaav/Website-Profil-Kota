Website profil statis yang menampilkan informasi lengkap tentang Kota Gresik, mencakup sejarah, budaya, pariwisata, kuliner khas, serta data demografi dan agenda kota. Dibangun dengan pendekatan modern, responsif, dan interaktif menggunakan HTML, CSS, dan JavaScript.

✨ Fitur
1. Desain Responsif – Tampilan optimal di semua perangkat (desktop, tablet, mobile)
2. Navigasi Interaktif – Menu responsif dengan toggle button dan smooth scroll
3. Slider Gambar Otomatis – Galeri foto dengan tombol navigasi, indikator, dan efek hover
4. Konten Dinamis – Daftar wisata dan kuliner di-generate secara otomatis via JavaScript
5. Tema Gelap Modern – Dark theme dengan aksen warna cyan dan ungu yang elegan
6. Sidebar Informasi – Profil daerah, agenda kota, dan data demografi
